# Event Center Website

A multi-page responsive website for an event center, built with HTML and CSS.

## Project Description
This website showcases an event center's offerings, schedule, and contact information. It includes a home page with featured events, an about page with venue information, an events page with a schedule table, and a contact form for inquiries.

## Features
- Responsive design (mobile and desktop)
- Navigation menu across all pages
- Featured events with images
- Event schedule table
- Contact form
- Social media links
- Hover effects and animations
- Back to top button
- Custom favicon

## Technologies Used
- HTML5 (semantic markup)
- CSS3 (Flexbox, Grid, animations)
- External stylesheet

## File Structure
- `index.html` - Home page
- `about.html` - About the venue
- `events.html` - Event listings with schedule table
- `contact.html` - Contact form
- `styles.css` - All styling
- `images/` - Image assets

## How to View
Open `index.html` in a web browser or use Live Server in VS Code.

## Requirements Met
✅ 3+ HTML pages with semantic markup
✅ Navigation, forms, tables, lists, images
✅ Internal, external, and anchor links
✅ Responsive CSS with Flexbox/Grid
✅ Hover/focus effects
✅ Custom button styles
✅ Animations and transitions
✅ Favicon
